Xenomai Lab
===========

A Platform for Digital Real-Time Control.

Installation
------------

A fairly standard procedure.

### Dependencies

`sudo apt-get install libboost-graph-dev qt4-dev-tools build-essential`

### Compile Xenomai Lab:

`qmake`

`make`

### Install

`sudo make install`

### Uninstall

`sudo make uninstall`
